## SARS-CoV-2 analysis with USA-focused subsampling
This analysis focuses on SARS-CoV-2 sequences available from the United States of America with even sampling across states and territories.

[MicrobeTrace](https://microbetrace.cdc.gov/MicrobeTrace/) edge lists for this tree can be downloaded from [https://github.com/CDCgov/spheres-auspice-data/tree/master/microbetrace](https://raw.githubusercontent.com/CDCgov/spheres-auspice-data/master/microbetrace/distance-USA.csv.xz).

[Auspice](https://auspice.us/) json files for this tree can be downloaded from [https://github.com/CDCgov/spheres-auspice-data/tree/master/auspice](https://raw.githubusercontent.com/CDCgov/spheres-auspice-data/master/auspice/ncov_USA.tar.xz).
